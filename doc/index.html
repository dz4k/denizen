denizen

Denizen is a pre-alpha work in progress. It lacks most of the features advertised here. You can see the progress on my to-do list.

Denizen is yet another blog CMS a personal website engine that connects you to the IndieWeb. No configuration, no connection strings, and hopefully for free.

Getstarted Source code Bug tracker Docs Proto­cols

Denizen is written in JavaScript for Deno, and can be deployed on Deno Deploy or on any server — lowering the barrier of having a personal website without the limitations of static blog generators.
