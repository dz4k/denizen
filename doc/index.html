Become a citizen of the world wide web

Denizen is a pre-alpha work in progress. You can see this progress on my to-do list.

Denizen is a personal website engine that connects you to the IndieWeb and Fediverse on your own terms. Find it on Codeberg.

> Get started

Denizen is written in JavaScript for Deno, and can be deployed on Deno Deploy — lowering the barrier of having a personal website without the limitations of static blog generators.

  • It comes with a post editor, so you can post from anywhere.
  • It supports Webmention (both receiving and sending) and ActivityPub, as well as on-site comments.
