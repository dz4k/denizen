The age of the social network is over and it's time to take back control of your online presence. You don't need a Content Management System, you need a place to post stuff — where you can sign up and start posting in seconds.

No website is an island. Your website connects with the IndieWeb and the Fediverse. That means you can follow and be followed on Denizen, Mastodon, RSS and an endless number of other places. Or not — turn off all interactions and scream into the void.

Available at cost. We use the Deno Deploy platform to host your website. That means you can get started for free, and as you scale up, we will charge you no more than what they charge us. (This means we can't offer tech support. If you're planning to use Denizen commercially, contact Deniz. Yes, I named the software after myself)

Open source software. All of Denizen's code is available online — not just a curated subset. That is how you know that your data is not going anywhere it shouldn't. We even release the code we use to run this hosting service, so you can start your own Denizen business.

Technical simplicity. Don't want to use our hosting? You can run Denizen on your own server or Deno Deploy account with a single command.
